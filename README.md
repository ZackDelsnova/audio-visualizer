# Audio Visualizer

A simple Rust program that plays an audio file and visualizes it in real time.

## Requirments
- Rust
- Desktop Development with C++

## Usage

``` bash
cargo run -- "<audiofile.mp3>" "<bg_color>" "<bar_color>"
```
